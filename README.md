## webpack template

一个支持多页面的 webpack 脚手架

适合于开发简单的 H5 页面或一些活动页

这里没有做 vue 和 react 的框架的支持，若需要引用，请另外添加 loader 支持

预览地址：https://churjan.github.io/webpack-template/dist/index.html
